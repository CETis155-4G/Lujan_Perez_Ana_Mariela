Html:

Hoy vimos lo que es Html, es un lenguaje de etiquetas que sirve para estructurar y desplegar una
pagina web.

Sus siglas significan "Hyper Text Markup Language",es decir, lenguaje de marcas de hipertexto
funciona a base de etiquetas escritas que aparecen entre comillas angulares, a partir de las cuales
se cifra la apariencia y orden interno de una pagina web.
